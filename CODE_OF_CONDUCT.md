# Code of Conduct

## Our Commitment

F1 is committed to a welcoming, respectful environment for everyone — regardless of skill level, background, or which AI agent they happen to use.

## Standards

**We encourage:** respectful interactions, constructive feedback, patience with newcomers, and a focus on improving the project for all users.

**We do not tolerate:** harassment, personal attacks, discriminatory comments, publishing private information, or any conduct that creates a hostile environment.

## Contributions

We value all forms of contribution: bug reports, code improvements, documentation, translations, screenshots, skill packs, and community support.

See [CONTRIBUTING.md](https://github.com/bastndev/F1/blob/main/CONTRIBUTING.md) for detailed guidelines.

## Enforcement

Report violations to **bastndev@gohit.xyz** using the subject format:

```
[F1] — Brief description
```

All reports are handled confidentially. Depending on severity, responses range from a private warning to permanent removal from the community.

## Scope

This Code of Conduct applies to all project spaces: GitHub issues, pull requests, discussions, and any public forum where this project is represented.
