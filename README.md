## ЁЯЪА Getting started

1. Press **`F1`** to open the **CLI Hub** panel.
2. Pick an agent from the launcher (or **Custom CLI** to run your own command).
3. Open **My Skills** from the activity bar (**`Ctrl+3`**) to install or create skills.

> To use (**`Alt + E`**) & (**`Ctrl + 3`**) , you need to have **[Lynx Keymap Pro](https://open-vsx.org/extension/bastndev/Lynx-Keymap)** installed.

<br>

<h2 align="center"> ЁЭЩ╡ЁЭЪОЁЭЪКЁЭЪЭЁЭЪЮЁЭЪЫЁЭЪОЁЭЪЬ </h2>

### ЁЯЦея╕П CLI Hub

- Run multiple AI coding CLIs side by side in embedded [xterm.js](https://xtermjs.org/) terminals.
- Fuzzy-search launcher to pick an agent; press **F1** to jump straight to the panel.
- Built-in tools alongside every session:
  - **Prompt** тАФ rich editor with `@`-file mentions, image paste, skill chips, live spellтАСmarking, and sourceтЖТEnglish translation before sending.
  - **Translator** тАФ translate any terminal selection inline.
  - **Use** тАФ perтАСCLI usage / status view.
  - **Keymaps** тАФ keyboardтАСshortcut reference.
  - **Commands** тАФ searchable, per-CLI palette of the active agent's slash-commands.
- **Voice** тАФ read replies aloud, plus an optional "ding" when an agent finishes while your attention is elsewhere.

### ЁЯзй My Skills

- Install skills from the marketplace тАФ **AllтАСtime**, **Trending (24h)**, **ЁЯФе Flame**, and **Official** sources.
- Create skills with guided generators for `AGENTS.md`, `CLAUDE.md`, and `DESIGN.md`, plus fast templates by category.
- Manage local and saved skills per workspace.

### ЁЯза Smart + Skills

- Flip the **Smart + Skills** toggle in the launcher, then pick an agent as usual.
- On launch, F1 builds a cheap code-graph of your project plus a set of built-in working rules, then has the agent read them тАФ so it starts already oriented and replies **"I am ready for work тЬЕ"** in the chat, with no manual context-pasting.
- Powered by **My Memory**: a fast, pure-TS engine that writes a structural `.f1/` project map and keeps each CLI's instruction file (`AGENTS.md` / `CLAUDE.md`) pointed at it. Scratch files are cleaned up automatically once the agent is primed.

Paste the following command and press `Enter`:

```bash

```
