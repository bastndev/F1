---

F1 превращает VS Code в хаб для AI-агентов программирования. Открывайте **Claude Code, Codex, Copilot, Cursor, Kiro, Kilo Code, OpenCode, Grok, Antigravity** — или любой пользовательский CLI — во встроенных терминалах, мгновенно переключайтесь между ними, пишите лучшие промпты и обеспечивайте недорогую загрузку контекста проекта для каждого агента.

<br>

## 🚀 Начало работы

1. Нажмите **`F1`**, чтобы открыть панель **CLI Hub**.
2. Выберите агента в лаунчере (или **Пользовательский CLI** для запуска собственной команды).
3. Откройте **My Skills** на панели активности (**`Ctrl+3`**), чтобы устанавливать или создавать навыки.

> Для использования (**`Alt + E`**) и (**`Ctrl + 3`**) необходимо установить **[Lynx Keymap Pro](https://open-vsx.org/extension/bastndev/Lynx-Keymap)**.

### 🖥️ CLI Hub

- Запускайте несколько CLI для AI-программирования бок о бок во встроенных терминалах [xterm.js](https://xtermjs.org/).
- Лаунчер с нечётким поиском для выбора агента; нажмите **F1** для перехода сразу к панели.
- Встроенные инструменты в каждой сессии:
  - **Промпт** — продвинутый редактор с упоминаниями файлов через `@`, вставкой изображений, чипами навыков, живой проверкой орфографии, историей промптов (`↑`), режимами **PRO / PLAN** и переводом с исходного на английский перед отправкой.
  - **Переводчик** — переводит любое выделение в терминале inline, с автоматическим режимом, который зачитывает ответ вслух, пока остальная часть ещё переводится.
  - **Использование** — просмотр использования/статуса по CLI.
  - **Карты клавиш** — справочник по горячим клавишам.
  - **Команды** — поисковая палитра слэш-команд активного агента по CLI.
- **Голос** — чтение ответов вслух, а также опциональный «динь», когда агент завершает работу, пока ваше внимание сосредоточено в другом месте.

### 🧩 My Skills

- Устанавливайте навыки из маркетплейса — источники **За все время**, **В тренде (24ч)**, **🔥 Flame** и **Официальные**.
- Создавайте навыки с помощью пошаговых генераторов для `AGENTS.md`, `CLAUDE.md` и `DESIGN.md`, а также быстрых шаблонов по категориям.
- Управляйте локальными и сохранёнными навыками для каждого рабочего пространства.

### 🧠 Smart + Skills

- Включите переключатель **Smart + Skills** в лаунчере, затем выберите агента как обычно.
- При запуске F1 строит недорогой граф кода вашего проекта и набор встроенных рабочих правил, затем заставляет агента их прочитать — так он начинает работу уже ориентированным и отвечает **"I am ready for work ✅"** в чате, без необходимости вручную вставлять контекст.
- На основе **My Memory**: быстрого движка на чистом TypeScript, который записывает структурную карту проекта `.f1/` и поддерживает файл инструкций каждого CLI (`AGENTS.md` / `CLAUDE.md`) указывающим на неё. Временные файлы автоматически удаляются после подготовки агента.

## Установка

### Способ 1 — Быстрое открытие

Запустите _Быстрое открытие_ в зависимости от вашей операционной системы:

- <img src="https://www.kernel.org/theme/images/logos/favicon.png" width=16 height=16/> Linux `Ctrl+P`
- <img src="https://developer.apple.com/favicon.ico" width=16 height=16/> macOS `⌘P`
- <img src="https://www.microsoft.com/favicon.ico" width=16 height=16/> Windows `Ctrl+P`

Вставьте следующую команду и нажмите `Enter`:

```bash
ext install bastndev.f1
```

### Способ 2 — Просмотр расширений

1. Откройте Расширения (`Ctrl+Shift+X` / `⌘+Shift+X`)
2. Поиск **"F1"** (издатель: `bastndev`)
3. Нажмите **Установить**
