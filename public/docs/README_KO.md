</div>

---

F1은 VS Code를 AI 코딩 에이전트의 허브로 변환합니다. **Claude Code, Codex, Copilot, Cursor, Kiro, Kilo Code, OpenCode, Grok, Antigravity** — 또는 맞춤 CLI를 내장 터미널에서 열고, 빠르게 전환하고, 더 나은 프롬프트를 작성하며, 각 에이전트가 저렴한 비용으로 프로젝트 컨텍스트를 로드할 수 있도록 유지합니다.

<br>

## 🚀 시작하기

1. **`F1`**을 눌러 **CLI Hub** 패널을 엽니다.
2. 런처에서 에이전트를 선택하세요 (또는 **맞춤 CLI**로 직접 명령을 실행할 수 있습니다).
3. 활동 표시줄에서 **My Skills** (**`Ctrl+3`**)를 열어 스킬을 설치하거나 만드세요.

> (**`Alt + E`**) 및 (**`Ctrl + 3`**)를 사용하려면 **[Lynx Keymap Pro](https://open-vsx.org/extension/bastndev/Lynx-Keymap)**가 설치되어 있어야 합니다.

<h2 align="center"> 𝙵𝚎𝚊𝚝𝚞𝚛𝚎𝚜 </h2>

### 🖥️ CLI Hub

- 내장 [xterm.js](https://xtermjs.org/) 터미널에서 여러 AI 코딩 CLI를 나란히 실행하세요.
- 검색 런처로 에이전트를 선택하고, **F1**을 눌러 패널로 바로 이동하세요.
- 모든 세션에 내장된 도구:
  -   - **Prompt** — `@`-파일 멘션, 이미지 붙여넣기, 스킬 칩, 실시간 맞춤법 표시, 프롬프트 기록(`↑`), **PRO / PLAN** 모드, 전송 전 소스→영어 번역을 지원하는 고급 편집기; 각 CLI 세션은 자체 초안과 열림/닫힘 상태를 유지합니다.
  - **Translator** — 터미널 선택 내용을 인라인으로 번역하며, 나머지가 아직 번역 중일 때 답변을 자동으로 읽어주는 자동 모드가 있습니다.
  - **Use** — CLI별 사용량 / 상태 뷰.
  - **Keymaps** — 키보드 단축키 참조.
  - **Commands** — 활성 에이전트의 슬래시 명령을 검색 가능한 CLI별 팔레트.
- - **Voice** — 답변을 큰 소리로 읽어주며, 에이전트가 완료될 때 "딩" 소리와 CLI가 결정(승인, 예/아니오, 선택)을 기다릴 때 뚜렷한 확인 신호를 제공합니다.
- - **Needs-input badge** — 탭 이름 옆에 깜빡이는 점(컴팩트 레이아웃의 아이콘 전용 배지)은 사용자를 기다리는 백그라운드 세션을 표시합니다.

### 🧩 My Skills

- 마켓플레이스에서 스킬을 설치하세요 — **전체 시간**, **인기 (24시간)**, **🔥 Flame**, 및 **공식** 소스.
- `AGENTS.md`, `CLAUDE.md`, 및 `DESIGN.md`용 안내 생성기로 스킬을 만들고, 카테고리별 빠른 템플릿을 사용하세요.
- 워크스페이스별 로컬 및 저장된 스킬을 관리하세요.

### 🧠 Smart + Skills

- 런처에서 **Smart + Skills** 토글을 켜고, 평소와 같이 에이전트를 선택하세요.
- F1은 시작 시 프로젝트의 저비용 코드 그래프와 내장 작동 규칙 세트를 구축한 다음, 에이전트가 이를 읽도록 하여 이미 방향을 잡은 상태로 시작하고 채팅에서 **"I am ready for work ✅"**라고 응답하며, 수동으로 컨텍스트를 붙여넣을 필요가 없습니다.
- **My Memory** 기반: 구조화된 `.f1/` 프로젝트 맵을 작성하고 각 CLI의 지침 파일(`AGENTS.md` / `CLAUDE.md`)을 이를 가리키도록 유지하는 빠른 순수 TypeScript 엔진입니다. 에이전트가 준비되면 임시 파일이 자동으로 정리됩니다.

## 설치

### 방법 1 — 빠른 열기

운영 체제에 따라 _빠른 열기_를 실행하세요:

- <img src="https://www.kernel.org/theme/images/logos/favicon.png" width=16 height=16/> Linux `Ctrl+P`
- <img src="https://developer.apple.com/favicon.ico" width=16 height=16/> macOS `⌘P`
- <img src="https://www.microsoft.com/favicon.ico" width=16 height=16/> Windows `Ctrl+P`

다음 명령을 붙여넣고 `Enter`를 누르세요:

```bash
ext install bastndev.f1
```

### 방법 2 — 확장 프로그램 뷰

1. 확장 프로그램 열기 (`Ctrl+Shift+X` / `⌘+Shift+X`)
2. **"F1"** 검색 (개발자: `bastndev`)
3. **설치** 클릭
