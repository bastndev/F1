**Starte und verwalte jede AI-Coding-CLI von einem VS Code-Panel aus.**

</div>

---

F1 verwandelt VS Code in ein Zentrum für AI-Coding-Agenten. Öffne **Claude Code, Codex, Copilot, Cursor, Kiro, Kilo Code, OpenCode, Grok, Antigravity** — oder jede eigene CLI — in eingebetteten Terminals, wechsle sofort zwischen ihnen, schreibe bessere Prompts und halte den Kontext deines Projekts kostengünstig für jeden Agenten ladbar.

<br>

## 🚀 Erste Schritte

1. Drücke **`F1`**, um das **CLI Hub**-Panel zu öffnen.
2. Wähle einen Agenten aus dem Launcher (oder **Custom CLI**, um eigenen Befehl auszuführen).
3. Öffne **My Skills** aus der Aktivitätenleiste (**`Ctrl+3`**), um Skills zu installieren oder zu erstellen.

> Zur Nutzung von (**`Alt + E`**) & (**`Ctrl + 3`**) muss **[Lynx Keymap Pro](https://open-vsx.org/extension/bastndev/Lynx-Keymap)** installiert sein.

<br>

<h2 align="center"> Funktionen </h2>

### 🖥️ CLI Hub

- Starte mehrere AI-Coding-CLIs nebeneinander in eingebetteten [xterm.js](https://xtermjs.org/)-Terminals.
- Fuzzy-Suche-Launcher zur Agentenauswahl; drücke **F1**, um direkt zum Panel zu springen.
- Integrierte Werkzeuge neben jeder Sitzung:
  - **Prompt** — Rich-Text-Editor mit `@`-Datei-Erwähnungen, Bild-Einfügung, Skill-Chips, Live-Rechtschreibmarkierung, Prompt-Verlauf (`↑`), **PRO / PLAN**-Modi und Quelle→Englisch-Übersetzung vor dem Senden.
  - **Translator** — Übersetze jede Terminal-Auswahl inline, mit einem automatischen Modus, der die Antwort laut vorliest, während der Rest noch übersetzt wird.
  - **Use** — CLI-spezifische Nutzung-/Status-Ansicht.
  - **Keymaps** — Tastaturkürzel-Referenz.
  - **Commands** — durchsuchbare, CLI-spezifische Palette der Slash-Befehle des aktiven Agenten.
- **Voice** — Antworten laut vorlesen, plus optionaler „Ding"-Ton, wenn ein Agent fertig ist, während deine Aufmerksamkeit woanders ist.

### 🧩 My Skills

- Installiere Skills aus dem Marketplace — **All-time**, **Trending (24h)**, **🔥 Flame** und **Official**-Quellen.
- Erstelle Skills mit geführten Generatoren für `AGENTS.md`, `CLAUDE.md` und `DESIGN.md`, plus schnelle Vorlagen nach Kategorie.
- Verwalte lokale und gespeicherte Skills pro Workspace.

### 🧠 Smart + Skills

- Aktiviere den **Smart + Skills**-Schalter im Launcher und wähle dann wie gewohnt einen Agenten aus.
- Beim Start baut F1 einen kostengünstigen Code-Graphen deines Projekts plus eine Reihe eingebauter Arbeitsregeln und lässt den Agenten diese lesen — so startet er bereits orientiert und antwortet **„Ich bin bereit für die Arbeit ✅"** im Chat, ohne manuelles Einfügen von Kontext.
- Unterstützt von **My Memory**: einer schnellen, reinen-TS-Engine, die eine strukturelle `.f1/`-Projektkarte schreibt und die Instruktionsdatei jeder CLI (`AGENTS.md` / `CLAUDE.md`) darauf zeigt. Temporäre Dateien werden automatisch bereinigt, sobald der Agent initialisiert ist.

## Installation

### Methode 1 — Schnell öffnen

Starte _Quick Open_ abhängig von deinem Betriebssystem:

- <img src="https://www.kernel.org/theme/images/logos/favicon.png" width=16 height=16/> Linux `Ctrl+P`
- <img src="https://developer.apple.com/favicon.ico" width=16 height=16/> macOS `⌘P`
- <img src="https://www.microsoft.com/favicon.ico" width=16 height=16/> Windows `Ctrl+P`

Füge den folgenden Befehl ein und drücke `Enter`:

```bash
ext install bastndev.f1
```

### Methode 2 — Erweiterungsansicht

1. Öffne Erweiterungen (`Ctrl+Shift+X` / `⌘+Shift+X`)
2. Suche nach **"F1"** (Herausgeber: `bastndev`)
3. Klicke auf **Install**
