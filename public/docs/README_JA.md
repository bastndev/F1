**1つの VS Code パネルからすべての AI コーディング CLI を起動および管理します。**

</div>

---

F1 は、VS Code を AI コーディングエージェントのハブに変えます。埋め込みターミナルで **Claude Code、Codex、Copilot、Cursor、Kiro、Kilo Code、OpenCode、Grok、Antigravity** — または任意のカスタム CLI — を開き、それらを瞬時に切り替え、より良いプロンプトを書き、すべてのエージェントがプロジェクトのコンテキストを安価に読み込めるようにします。

<br>

## 🚀 はじめに

1. **`F1`** を押して **CLI Hub** パネルを開きます。
2. ランチャーからエージェントを選択します（または **Custom CLI** で独自のコマンドを実行します）。
3. アクティビティバー（**`Ctrl+3`**）から **My Skills** を開いて、スキルをインストールまたは作成します。

> (**`Alt + E`**) と (**`Ctrl + 3`**) を使用するには、**[Lynx Keymap Pro](https://open-vsx.org/extension/bastndev/Lynx-Keymap)** をインストールする必要があります。

<br>

<h2 align="center"> 機能 </h2>

### 🖥️ CLI Hub

- 埋め込み [xterm.js](https://xtermjs.org/) ターミナルで、複数の AI コーディング CLI を並べて実行します。
- エージェントを選択するためのファジー検索ランチャー。**F1** を押すとパネルに直接ジャンプします。
- 各セッションの横に組み込まれたツール:
  - **Prompt** — `@`ファイルメンション、画像の貼り付け、スキルチップ、ライブスペルチェック、送信前のソース→英語翻訳を備えたリッチエディター。
  - **Translator** — ターミナルの選択範囲をインラインで翻訳します。
  - **Use** — CLI ごとの使用状況 / ステータスビュー。
  - **Keymaps** — キーボードショートカットの参照。
  - **Voice** — 返信を読み上げます。さらに、別の作業中にエージェントが終了したときのオプションの「ディン」音。

### 🧩 My Skills

- マーケットプレイスからスキルをインストール — **All‑time**、**Trending (24h)**、**🔥 Flame**、**Official** ソース。
- `AGENTS.md`、`CLAUDE.md`、`DESIGN.md` のガイド付きジェネレーター、およびカテゴリ別の高速テンプレートを使用してスキルを作成します。
- ワークスペースごとにローカルおよび保存されたスキルを管理します。

### 🧠 My Memory

- コミットされた `.f1/` プロジェクトコンテキストマップを生成して、どの CLI も安価な共有コンテキストで起動できるようにします。
- プロジェクトごとのトグル。命令ファイル（`AGENTS.md` / `CLAUDE.md`）はコンテキストを指したままになります。

## インストール

### 方法 1 — クイックオープン

オペレーティングシステムに応じて _Quick Open_ を起動します:

- <img src="https://www.kernel.org/theme/images/logos/favicon.png" width=16 height=16/> Linux `Ctrl+P`
- <img src="https://developer.apple.com/favicon.ico" width=16 height=16/> macOS `⌘P`
- <img src="https://www.microsoft.com/favicon.ico" width=16 height=16/> Windows `Ctrl+P`

次のコマンドを貼り付けて `Enter` を押します:

```bash
ext install bastndev.f1
```

### 方法 2 — 拡張機能ビュー

1. 拡張機能を開きます（`Ctrl+Shift+X` / `⌘+Shift+X`）
2. **"F1"** を検索します（パブリッシャー：`bastndev`）
3. **インストール** をクリックします
