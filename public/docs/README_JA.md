**1つのVS CodeパネルからすべてのAIコーディングCLIを起動・管理します。**

</div>

---

F1はVS CodeをAIコーディングエージェントのハブに変えます。埋め込みターミナルで **Claude Code、Codex、Copilot、Cursor、Kiro、Kilo Code、OpenCode、Grok、Antigravity** — またはカスタムCLI を開き、瞬時に切り替え、より良いプロンプトを作成し、各エージェント用にプロジェクトのコンテキストを低コストで維持します。

<br>

## 🚀 はじめに

1. **CLI Hub** パネルを開くには **`F1`** を押します。
2. ランチャーからエージェントを選択するか（または独自のコマンドを実行するための **カスタムCLI**）。
3. スキルをインストールまたは作成するには、アクティビティバー（**`Ctrl+3`**）から **My Skills** を開きます。

> (**`Alt + E`**) & (**`Ctrl + 3`**) を使用するには、**[Lynx Keymap Pro](https://open-vsx.org/extension/bastndev/Lynx-Keymap)** がインストールされている必要があります。

<br>

<h2 align="center"> 𝙵𝚎𝚊𝚝𝚞𝚛𝚎𝚜 </h2>

### 🖥️ CLI Hub

- 埋め込み [xterm.js](https://xtermjs.org/) ターミナルで複数のAIコーディングCLIを並べて実行します。
- エージェントを選択するためのファジーサーチランチャー；パネルに直接移動するには **F1** を押します。
- 各セッションに組み込みツール：
  - **Prompt** — `@`ファイルメンション、画像貼り付け、スキルチップ、ライブスペルチェック、プロンプト履歴（`↑`）、**PRO / PLAN** モード、送信前のソース→英語翻訳を備えたリッチエディタ。
  - **Translator** — 任意のターミナル選択をインライン翻訳し、残りがまだ翻訳中に応答を読み上げるオートモード付き。
  - **Use** — CLIごとの使用量 / ステータスビュー。
  - **Keymaps** — キーボードショートカットリファレンス。
  - **Commands** — アクティブエージェントのスラッシュコマンドの検索可能なCLIごとのパレット。
- **Voice** — 応答を読み上げ、エージェントが他の作業中に完了した場合のオプションの「ディング」音。

### 🧩 My Skills

- マーケットプレイスからスキルをインストール — **全期間**、**トレンド（24h）**、**🔥 Flame**、**公式**ソース。
- `AGENTS.md`、`CLAUDE.md`、`DESIGN.md` のガイド付きジェネレーターでスキルを作成し、カテゴリ別のテンプレートも用意。
- ワークスペースごとにローカルおよび保存済みスキルを管理します。

### 🧠 Smart + Skills

- ランチャーの **Smart + Skills** トグルを切り替え、通常通りエージェントを選択します。
- 起動時、F1はプロジェクトの低成本コードグラフと組み込みルールセットを構築し、エージェントにこれらを読ませます — これにより、すでに方向づけられた状態で起動し、チャットで**「作業の準備ができました ✅」**と応答し、手動でのコンテキスト貼り付けは不要です。
- **My Memory** によって駆動：プロジェクトの構造的な `.f1/` マップを書き、各CLIの指示ファイル（`AGENTS.md` / `CLAUDE.md`）を它に向ける高速な純TypeScriptエンジン。エージェントの準備が完了すると一時ファイルは自動的にクリーンアップされます。

## インストール

### 方法1 — クイックオープン

お使いのオペレーティングシステムに応じて _クイックオープン_ を起動します：

- <img src="https://www.kernel.org/theme/images/logos/favicon.png" width=16 height=16/> Linux `Ctrl+P`
- <img src="https://developer.apple.com/favicon.ico" width=16 height=16/> macOS `⌘P`
- <img src="https://www.microsoft.com/favicon.ico" width=16 height=16/> Windows `Ctrl+P`

以下のコマンドを貼り付けて `Enter` を押します：

```bash
ext install bastndev.f1
```

### 方法2 — 拡張機能ビュー

1. 拡張機能を開く（`Ctrl+Shift+X` / `⌘+Shift+X`）
2. **"F1"** を検索（パブリッシャー：`bastndev`）
3. **インストール** をクリック
