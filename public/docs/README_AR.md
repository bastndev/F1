**شغّل وأدر كل واجهة سطر أوامر للبرمجة بالذكاء الاصطناعي من لوحة واحدة في VS Code.**

</div>

---

يحوّل F1 VS Code إلى مركز ل وكلاء البرمجة بالذكاء الاصطناعي. افتح **Claude Code, Codex, Copilot, Cursor, Kiro, Kilo Code, OpenCode, Grok, Antigravity** — أو أي واجهة سطر أوامر مخصصة — في طرفيات مدمجة، والتبديل بينها فوراً، وكتابة مطالبات أفضل، والحفاظ على سياق مشروعك بتكلفة أقل تحميلاً لكل وكيل.

<br>

## 🚀 البدء

1. اضغط **`F1`** لفتح لوحة **CLI Hub**.
2. اختر وكيلاً من المُشغّل (أو **واجهة سطر أوامر مخصصة** لتشغيل أمرك الخاص).
3. افتح **My Skills** من شريط النشاط (**`Ctrl+3`**) لتثبيت أو إنشاء مهارات.

> لاستخدام (**`Alt + E`**) و (**`Ctrl + 3`**)، يجب عليك تثبيت **[Lynx Keymap Pro](https://open-vsx.org/extension/bastndev/Lynx-Keymap)**.

<h2 align="center"> المميزات </h2>

### 🖥️ CLI Hub

- شغّل عدة واجهات سطر أوامر للبرمجة بالذكاء الاصطناعي جنباً إلى جنب في طرفيات [xterm.js](https://xtermjs.org/) المدمجة.
- مُشغّل بحث ضبابي لاختيار وكيل؛ اضغط **F1** للانتقال مباشرة إلى اللوحة.
- أدوات مدمجة مع كل جلسة:
  -   - **Prompt** — محرر متقدم مع ذكر الملفات بـ `@`، ولصق الصور، وشرائح المهارات، وتحديد الأخطاء الإملائية مباشرة، وسجل المطالبات (`↑`)، وأوضاع **PRO / PLAN**، وترجمة المصدر→الإنجليزية قبل الإرسال؛ كل جلسة CLI تحتفظ بمسودتها الخاصة وحالتها المفتوحة/المغلقة.
  - **المترجم** — ترجم أي تحديد من الطرفية مباشرةً، مع وضع تلقائي يقرأ الإجابة بصوت عالٍ بينما يتواصل تحويل الباقي؛ صغّره للحفاظ على أدوات التشغيل في رأس CLI.
  - **Use** — عرض الاستخدام/الحالة لكل واجهة سطر أوامر.
  - **Keymaps** — مرجع اختصارات لوحة المفاتيح.
  - **Commands** — لوحة قابلة للبحث من أوامر الشرط النشطة لكل واجهة سطر أوامر.
- **الصوت** — اقرأ الردود بصوت عالٍ مع أدوات إيقاف مؤقت وإرجاع واستئناف دائمة، إضافةً إلى صوت "طنين" عند انتهاء الوكيل وإشارة تأكيد مميزة عندما يكون CLI في انتظار قرار (موافقة، نعم/لا، اختيار).
- - **Needs-input badge** — نقطة نابضة بجانب اسم التبويب (شارة أيقونة فقط في التخطيط المضغوط) تشير إلى أي جلسة خلفية تنتظر منك.

### 🧩 My Skills

- ثبّت المهارات من السوق — مصادر **الكل‑الوقت**، **الرائجة (24 ساعة)**، **🔥 الشعلة**، و**الرسمية** — مع إمكانية إلغاء التنزيل وتقدم التثبيت.
- أنشئ مهارات مع مولّدات موجّهة لـ `AGENTS.md` و`CLAUDE.md` و`DESIGN.md`، بالإضافة إلى قوالب سريعة حسب الفئة.
- أدر المهارات المحلية والمحفوظة لكل مساحة عمل.

### 🧠 Smart + Skills

- فعّل مفتاح **Smart + Skills** في المُشغّل، ثم اختر وكيلاً كالمعتاد.
- عند التشغيل، يقوم F1 ببناء خريطة كود رخيصة التكلفة لمشروعك مع مجموعة من قواعد العمل المدمجة، ثم يجعل الوكيل يقرأها — فيبدأ موجهاً مسبقاً ويُرد **"أنا جاهز للعمل ✅"** في المحادثة، دون الحاجة إلى نسخ السياق يدوياً.
- مدعوم بـ **My Memory**: محرك سريع بالـ TypeScript الكامل يكتب خريطة هيكلية للمشروع في `.f1/` ويُبقي ملف تعليمات كل واجهة سطر أوامر (`AGENTS.md` / `CLAUDE.md`) مُوجّهاً إليها. يتم حذف الملفات المؤقتة تلقائياً بمجرد تجهيز الوكيل.

## التثبيت

### الطريقة 1 — الفتح السريع

شغّل _Quick Open_ حسب نظام التشغيل الخاص بك:

- <img src="https://www.kernel.org/theme/images/logos/favicon.png" width=16 height=16/> Linux `Ctrl+P`
- <img src="https://developer.apple.com/favicon.ico" width=16 height=16/> macOS `⌘P`
- <img src="https://www.microsoft.com/favicon.ico" width=16 height=16/> Windows `Ctrl+P`

الصق الأمر التالي واضغط `Enter`:

```bash
ext install bastndev.f1
```

### الطريقة 2 — عرض الإضافات

1. افتح الإضافات (`Ctrl+Shift+X` / `⌘+Shift+X`)
2. ابحث عن **"F1"** (الناشر: `bastndev`)
3. اضغط **Install**
