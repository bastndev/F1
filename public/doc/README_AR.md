**قم بتشغيل وإدارة كل واجهات سطر الأوامر الخاصة ببرمجة الذكاء الاصطناعي من لوحة واحدة في VS Code.**

</div>

---

يجعل F1 من VS Code مركزاً لوكلاء برمجة الذكاء الاصطناعي. افتح **Claude Code، Codex، Copilot، Cursor، Kiro، Kilo Code، OpenCode، Grok، Antigravity** — أو أي واجهة سطر أوامر مخصصة — في أجهزة طرفية مضمنة، وقم بالتبديل بينها على الفور، واكتب مطالبات أفضل، وحافظ على سياق مشروعك خفيفاً للتحميل لكل وكيل.

## ⌨️ اختصارات لوحة المفاتيح

| الأمر                   | 🍎 macOS            | 🟦 Windows             | 🐧 Linux               | مدعوم |
| :------------------------ | :------------------ | :--------------------- | :--------------------- | --------- |
| التركيز على CLI             | `F1`                | `F1`                   | `F1`                   | ㅤㅤ✅    |
| التركيز على المهارات        | `⌘ + 3`             | `Ctrl + 3`             | `Ctrl + 3`             | ㅤㅤ✅    |

| -                         | -                   | -                      | -                      |           |
| اللوحة الجانبية (يمين/يسار)          | `⌘ + CapsLock`      | `Ctrl + CapsLock`      | `Ctrl + CapsLock`      | ㅤㅤ✅    |

<h2 align="center">:: الميزات ::</h2>

### 🖥️ CLI Hub

- تشغيل واجهات برمجة متعددة للذكاء الاصطناعي جنباً إلى جنب في أجهزة طرفية مضمنة [xterm.js](https://xtermjs.org/).
- مشغل بحث ضبابي لاختيار وكيل؛ اضغط على **F1** للانتقال مباشرة إلى اللوحة.
- أدوات مدمجة بجانب كل جلسة:
  - **المطالبة (Prompt)** — محرر غني بذكر الملفات `@`، ولصق الصور، ورقاقات المهارات، وعلامات التدقيق الإملائي المباشر، والترجمة من المصدر→الإنجليزية قبل الإرسال.
  - **المترجم (Translator)** — ترجم أي تحديد في الجهاز الطرفي مدمجاً.
  - **الاستخدام (Use)** — عرض الاستخدام / الحالة لكل CLI.
  - **خرائط المفاتيح (Keymaps)** — مرجع اختصارات لوحة المفاتيح.
- **الصوت (Voice)** — قراءة الردود بصوت عالٍ، بالإضافة إلى رنين اختياري "ding" عند انتهاء وكيل بينما يكون انتباهك في مكان آخر.

### 🧩 مهاراتي (My Skills)

- تثبيت المهارات من المتجر — مصادر **كل الوقت (All-time)**، **الرائجة (24h)**، **🔥 المشتعلة (Flame)**، و **الرسمية (Official)**.
- إنشاء المهارات باستخدام مولدات موجهة لـ `AGENTS.md` و `CLAUDE.md` و `DESIGN.md`، بالإضافة إلى قوالب سريعة حسب الفئة.
- إدارة المهارات المحلية والمحفوظة لكل مساحة عمل.

### 🧠 ذاكرتي (My Memory)

- إنشاء خريطة سياق مشروع `.f1/` ملتزمة بحيث يبدأ أي CLI بسياق مشترك ورخيص.
- تبديل لكل مشروع؛ تظل ملفات التعليمات (`AGENTS.md` / `CLAUDE.md`) موجهة إلى السياق.

## 🚀 البدء

1. قم بتثبيت **F1** من [VS Code Marketplace](https://marketplace.visualstudio.com/items?itemName=bastndev.f1).
2. اضغط على **`F1`** لفتح لوحة **CLI Hub**.
3. اختر وكيلاً من المشغل (أو **Custom CLI** لتشغيل أمرك الخاص).
4. افتح **My Skills** من شريط النشاط (**`Ctrl+3`**) لتثبيت المهارات أو إنشائها.

> يؤدي التشغيل الأول لأي وكيل إلى تثبيت CLI الخاص به إذا لم يكن موجوداً بالفعل في `PATH`.

## التثبيت

### الطريقة 1 — الفتح السريع (Quick Open)

قم بتشغيل _Quick Open_ اعتمادًا على نظام التشغيل الخاص بك:

- <img src="https://www.kernel.org/theme/images/logos/favicon.png" width=16 height=16/> Linux `Ctrl+P`
- <img src="https://developer.apple.com/favicon.ico" width=16 height=16/> macOS `⌘P`
- <img src="https://www.microsoft.com/favicon.ico" width=16 height=16/> Windows `Ctrl+P`

الصق الأمر التالي واضغط على `Enter`:

```bash
ext install bastndev.f1
```

### الطريقة 2 — عرض الإضافات

1. افتح الإضافات (`Ctrl+Shift+X` / `⌘+Shift+X`)
2. ابحث عن **"F1"** (الناشر: `bastndev`)
3. انقر فوق **تثبيت (Install)**
